# Digimax

Enterprise Digital Signage Engine.

## Objectives

Digimax is a modular digital signage platform designed for:

* Android
* Linux
* Windows
* WebOS
* Tizen

Android is the first implementation target.

## Core Features

* Scheduler Engine
* Content Management
* Playback Engine
* Programmatic Advertising
* Device Monitoring
* Security Platform
* Cluster Election
* OTA Update

## Architecture Principles

* Offline First
* Security First
* OS Agnostic
* Configuration Driven
* Cluster Ready
* Rust Ready
